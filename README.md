# Using-Mediapipe-project

# 🎯 MediaPipe Computer Vision Projects Collection

A complete collection of **13+ Real-Time Computer Vision Projects** built using **Python, OpenCV, and Google MediaPipe**.

This repository demonstrates various AI-powered real-time detection systems including face detection, hand tracking, pose estimation, object detection, motion tracking, face transformation, and more.

✔ Hand Landmark Detection

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/MediaPipe-Projects.git
```

Go to project folder

```bash
cd MediaPipe-Projects
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run any project

```bash
python filename.py
```

# 📦 Requirements
